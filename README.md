<h2>Setup Bot</h2>

1. Create `.env` file
2. Put in `.env` 
```shell
BOT_TOKEN=
MONGODB_URL=
WEBHOOK_GUILDS_UPDATE=
```
3. Configure `my_utils/consts.py` file
3. Run bot via opening `main.py` file
